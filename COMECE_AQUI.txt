🚀 COMEÇAR EM 3 PASSOS

  1️⃣  Instalar dependências:
      pip install -r requirements.txt

  2️⃣  Editar .env com sua API key (football-data.org):

  3️⃣  Iniciar a aplicação:

✨ NOVAS FUNCIONALIDADES v3.0

  ✅ Página Comparador (/comparador)
     - Compara 2 times lado a lado
     - Skeleton loaders enquanto carrega
     
  ✅ Página Rankings (/ranking/pontos)
     - 3 tipos de ranking: Pontos, Gols, Defesa
     - Gráficos com Chart.js
     - Tabela responsiva
     
  ✅ API REST Endpoints
     - GET/POST/DELETE /api/favoritos
     - POST /api/comparar
     - GET /api/ranking
     
  ✅ Animações CSS (20+ efeitos)
     - slideInUp, shimmer, pulse, ripple, flip, bounce, glow, etc
     
  ✅ Utilities Library (utilities.js)
     - 20+ funções reutilizáveis
     - Toast notifications, modals, lazy loading, etc
     
  ✅ PWA (Progressive Web App)
     - Instalável em celular
     - Funciona offline
     
  ✅ Dark Mode
     - Persiste no localStorage
     - Smooth transitions
